# مشروع كتاب: من عبدالرحمن النقيدان إلى رويدا 🌙

كتاب شهر العسل — يحوّل تسجيل «recap» صوتيًّا من رحلة بالي إلى **مذكّرات روائية**
بصوت عبدالرحمن، مكتوبة من قلبه إلى رويدا.

## المنظومة (خمس طبقات)

```
① الذاكرة      ماذا حدث؟ وما الدليل؟        docs/01_relationship_knowledge_base.md
② البيانات     ماذا تقول الأرقام؟           analysis/  (سكربت + تقرير)
③ العقل        كيف تعمل العلاقة؟ ولماذا؟    docs/03_relationship_intelligence_engine.md
④ الخطة        كيف تتحول إلى فصول؟          docs/04_master_book_plan.md
⑤ الكتاب       المنتج النهائي               book/
```

> الطبقة ⓪ التأسيسية (تحليل المصادر + دستور الصوت والحقائق): `docs/00_book_bible.md`.

## جلب الصور والفيديوهات من Google Drive

**الطريقة الأسرع** (تحميل الناقص فقط — ٨ ملفات بالتوازي):
```bash
pip install gdown
sudo apt-get install -y parallel   # مرة واحدة
export PATH="$HOME/.local/bin:$PATH"

python3 scripts/download_drive_media.py --mode fast --workers 8
```

**أسرع من كل شيء:** من المتصفح → افتح [المجلد](https://drive.google.com/drive/folders/1OkFMZtF3dJE8xOehDp4avaHJZpJ-rEnM) → كليك يمين → **تنزيل** (ZIP واحد).

المجلد: **١٥٢٠ ملف** (١٣٥٧ صورة + ١٦٣ فيديو).

## تشغيل التحليل الكمّي

```bash
python3 analysis/analyze_conversation.py
```

## الحقائق الثابتة (مرجع سريع)
- الزوج **عبدالرحمن النقيدان** · الزوجة **رويدا** (من الرياض). التهجئة المعتمدة: **رويدا**.
- المسار: الرياض ← الدوحة ← (عبور بانكوك) ← بالي. الإقامة: **أوبود** ثم **نوسا دوا**.
- الطيران: الخطوط القطرية · فندق نوسا دوا: ريتز كارلتون · مطعم أوبود: Dona.
- التسجيل: السبت ٦ يونيو ٢٠٢٦، اليوم الرابع، موجّه لأطفال المستقبل.
